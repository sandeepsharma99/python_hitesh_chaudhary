- Python is an **interpreted** programming language  
- It is **dynamically typed**
- It follows **object-oriented** programming principles
- Python is a **high-level** language with **simple and readable syntax**
- It has a **vast standard library** and strong **package support**
- It is **case-sensitive** (e.g., `Apple` and `apple` are different)
- Python enables **rapid application development**
- It is widely used in:
  - Data Science and Analytics
  - Application Development
  - Game Development
  - Artificial Intelligence (AI) and Machine Learning (ML)
